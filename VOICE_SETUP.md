# 🎙️ ElevenLabs 음성 설정 가이드

마음 정원에 진짜 사람 목소리를 넣으려면 ElevenLabs에서 캐릭터별 음성을 고르고
Voice ID를 코드에 입력한 뒤, 음성 생성 스크립트를 한 번 실행하면 됩니다.

> 음성을 설정하지 않아도 게임은 정상 동작합니다(자막으로 폴백). 음성은 선택사항입니다.

---

## 1단계: 음성 고르기

1. https://elevenlabs.io 로그인
2. 왼쪽 메뉴 → **Voices** → **Voice Library**
3. 필터: **Language = Korean**
4. 각 캐릭터에 맞는 음성을 시청한 뒤 선택

## 2단계: Voice ID 복사

1. 마음에 드는 음성 → `...` 메뉴 → **Copy Voice ID**
2. 형태 예시: `21m00Tcm4TlvDq8ikWAM` (영문+숫자 20자)

## 3단계: 코드에 입력

`src/data/voiceConfig.ts` 파일을 열고, 각 캐릭터의
`PLACEHOLDER_XXX` 를 복사한 Voice ID로 교체합니다.

```ts
solomon: {
  ...
  voiceId: '21m00Tcm4TlvDq8ikWAM',  // ← PLACEHOLDER_SOLOMON 에서 교체
  ...
}
```

> 일부만 교체해도 됩니다. PLACEHOLDER로 남은 캐릭터는 음성 생성에서 자동으로 건너뜁니다.

## 4단계: API 키 입력

프로젝트 루트의 `.env` 파일(없으면 `.env.example` 복사해서 생성)에:

```
ELEVENLABS_API_KEY=sk_여기에_본인_키
```

> ⚠️ `.env`는 `.gitignore`에 포함되어 GitHub에 올라가지 않습니다. 키를 코드에 하드코딩하지 마세요.

## 5단계: 음성 생성 실행

```bash
# 예상 크레딧 미리 확인
npm run voices:count

# 실제 음성 생성 (public/voices/*.mp3 로 저장)
npm run voices:generate
```

- 이미 생성된 파일은 자동으로 건너뜁니다(캐싱 → 크레딧 절약).
- 대본을 바꾼 클립만 다시 생성하려면 해당 mp3를 지우고 다시 실행하세요.

## 캐릭터별 추천 음성 특징

| 캐릭터 | 추천 톤 |
|--------|---------|
| 솔로몬 부엉이 | 중후한 남성, 50~60대, 지혜롭고 따뜻함 |
| 세종 거북이 (받아쓰기) ⭐ | 발음 또렷한 남성/여성 — **가장 중요** |
| 백조 공주 | 우아하고 부드러운 여성 |
| 번개 토끼 | 활기찬 젊은 목소리 |
| 모차르트 다람쥐 | 밝고 경쾌함 |
| 시간 부엉이 | 지적이고 차분한 학자 |
| 호기심 여우 | 발랄하고 친근함 |
| 숫자 곰돌이 | 느긋하고 다정하지만 논리적 |
| 위로 토끼 | 부드럽게 위로하는 톤 |
| 내레이터 | 명확하고 친근한 표준 안내 |

⭐ **세종 거북이**는 국어 도장 받아쓰기에 쓰이므로 발음 정확성이 가장 중요합니다.
